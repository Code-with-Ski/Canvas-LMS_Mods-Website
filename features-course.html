Course Mods

The following features are mods to areas within a Canvas course.

  • Global Course Mods
    • Make course header sticky
  • Course Home Page
    • Add course statistics button
  • Modules
    • Add "Jump to Module" feature
  • People/Users
    • Add button to export users to CSV*#
    • Add hide inactive users feature
    • Add section filter*
    • Add column sort buttons
  • Groups
    • Add button to export groups to CSV*#
  • User
    • Add button to export access report to CSV
  • SpeedGrader
    • Convert text resembling links (begins with https:// or https://) to a hyperlink in the comments
    • Change the draft comment indicator from "*" to "DRAFT"
  • Statistics
    • Add course reports*
  • Assignments
    • Add button to load assignment details to list of assignments*
    • Include Turnitin check with assignment details*
    • Add button to export grades on assignment page*
    • Add rubric used for grading indicator on assignment page*
  • Discussions
    • Add button to export grades on discussion page*
  • Rubrics
    • Add drag-and-drop criteria rows feature
    • Add import rubric criteria feature*
    • Add rubric associations report on rubric page*
  • Sections
    • Add section reports*

* - Uses the Canvas LMS REST API
# - Requires course admin permissions
