Admin Mods

The following features are mods to areas within a root account/sub-account which are only accessible by Canvas admins.

  • Global Navigation Admin Menu
    • Add quick access admin links (course search, people search, and some account links)*
  • Course Search
    • Prevent input for "Show only blueprint courses" from filling to the end of the page in desktop view
    • Add additional search inputs (published/unpublished, sort by Canvas course ID)
    • Add the course code to the course search results*
    • Add concluded icon to course names in search results that are concluded*
    • Convert subaccount name in search results to link*$
    • Convert number of students in search results to link to People in course
    • Add a "View Grades" button to the course search results
  • People
    • User Account Details
      • Course Enrollments Box
        • Set box to be resizable
        • Set a default height
        • Sort enrollments by status, term, and role
        • Add enrollment filters (course published status, enrollment status, and term)
        • Add course code*
        • Add Canvas course ID
      • Accounts Box
        • Set box to be resizable
        • Set a default height
        • Show the admin roles in each account*
      • Groups Box
        • Set box to be resizable
        • Set a default height
      • Avatar Image
        • Set to be resizable by dragging the corner and/or clicking it
      • Add link to grades for active courses
    • User Grades
      • Update name and headings to reflect the user*
    • Profile Pictures
      • Set images to be resizable by dragging the corner
      • Adjust style to be rounded square instead of circle to show more of the background
      • Change the default image size
  • Rubrics
    • Add a search feature
    • Add drag-and-drop criteria rows feature
    • Add import rubric criteria feature*
    • Add rubric associations report on rubric page*
  • Question Banks
    • Add a search feature
  • Sub-accounts
    • Show Canvas account ID
    • Show SIS account ID*
  • Terms
    • Show the Canvas Term ID
    • [DEPRECATED] Add a search feature
    • [DEPRECATED] Move the add term button to the top
  • SIS Import
    • Add SIS History Log*

* - Uses the Canvas LMS REST API
$ - Requires manage account settings permission
